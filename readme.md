# Reclaim-Fuel integration

## Build
```bash
forc build
```

## Test
```bash
forc test
```

## Deploy
```bash
forc deploy --testnet
```

## Frontend
```bash
cd frontend 
npm i --legacy-peer-deps
```

Run the project:
```bash
npm run dev
```
